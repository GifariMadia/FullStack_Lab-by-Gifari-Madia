# FullStack_Lab

## Struktur Folder
1. **Projects/** - Semua proyek latihan.
2. **Snippets/** - Kode pendek atau kamus coding.
3. **Resources/** - Dokumentasi, buku, video, dan tautan penting.
4. **Notes/** - Catatan belajar pribadi.
5. **Tools/** - Konfigurasi dan skrip otomatisasi.
6. **Templates/** - Template proyek yang sering digunakan.

## Cara Penggunaan
- Tambahkan setiap proyek baru ke folder `Projects/`.
- Gunakan `Snippets/` untuk menyimpan solusi kode atau trik.
- Gunakan `Resources/` untuk referensi belajar.
